## ページ構成

1. **TOP** - ビジョン、会社紹介、数字で見るネクストステージ、CTA
2. **会社を知る** - 代表メッセージ、ミッション・ビジョン、沿革、会社概要
3. **仕事を知る** - 職種紹介、1日の流れ、キャリアパス
4. **社員を知る** - 社員インタビュー（3〜5名）
5. **働く環境** - 福利厚生、オフィス、カルチャー
6. **募集要項** - 職種別募集要項、選考フロー
7. **エントリー** - エントリーフォーム、FAQ
